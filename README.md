# SamShell
A linux shell made as a hobby project to learn C and lower-level programming. This repository was made possible thanks to CodeCrafter's awesome platform at https://codecrafters.io/ !

# Usage
SamShell provides the builtin exit, echo, type, pwd, and cd (abs paths only) commands, as well as the ability to run any executable in your system's PATH.

# Comments
I build this shell over the course of 4 days and learned a lot about software. I didn't have an LSP when I made it, just tree-sitter parsing and segmentation faults :). Looking back, I spotted a double free at the exit code. It never actually crashed, so maybe it's not a problem (yet). I had a lot of fun writing this, and I lost a lot of fear over software development; hopefully I'll find time to continue this project. As of now, however, this repository's purpose is to display my C skills rather than offer anything particularly useful. I'll decide on a proper license if the project ever grows into anything beyond a display.
